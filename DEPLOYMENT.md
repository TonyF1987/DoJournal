# 部署指南

本文档详细介绍如何将儿童作业打卡小程序部署到微信小程序平台。

## 前置条件

1. 已注册微信小程序账号
2. 已开通微信云开发
3. 已完成代码开发

## 部署步骤

### 一、小程序账号配置

#### 1.1 登录微信公众平台

访问 [微信公众平台](https://mp.weixin.qq.com/)，使用管理员账号登录。

#### 1.2 设置小程序信息

1. 进入「设置」->「基本设置」
2. 填写小程序名称（如：作业打卡）
3. 上传小程序图标（建议 512×512 像素）
4. 填写小程序简介
5. 选择服务类目：教育 -> 在线教育

#### 1.3 配置服务器域名

由于使用云开发，无需配置服务器域名。

### 二、云开发环境配置

#### 2.1 开通云开发

1. 在微信开发者工具中，点击顶部「云开发」按钮
2. 点击「开通」
3. 选择「免费版」套餐
4. 创建环境，输入环境名称（如：homework-checkin）
5. 记下环境ID（格式：`xxx-xxx`）

#### 2.2 配置环境ID

打开 `app.js` 文件，修改环境ID：

```javascript
wx.cloud.init({
  env: 'your_env_id', // 替换为你的环境ID
  traceUser: true
});
```

### 三、数据库初始化

#### 3.1 创建数据库集合

1. 在微信开发者工具中，点击顶部「云开发」
2. 进入「数据库」
3. 点击「+」创建集合，依次创建以下4个集合：
   - `users`
   - `homework`
   - `rewards`
   - `exchange_records`

#### 3.2 创建数据库索引

在云开发控制台的「数据库」->「索引管理」中添加索引：

**users 集合索引：**
```json
{
  "字段": "_openid",
  "类型": "文本",
  "属性": "唯一"
}
```

**homework 集合索引：**
```json
[
  {
    "字段": "_openid",
    "类型": "文本"
  },
  {
    "字段": "status",
    "类型": "文本"
  },
  {
    "字段": "createTime",
    "类型": "日期",
    "排序": "降序"
  }
]
```

**rewards 集合索引：**
```json
{
  "字段": "points",
  "类型": "数字",
  "排序": "升序"
}
```

**exchange_records 集合索引：**
```json
[
  {
    "字段": "_openid",
    "类型": "文本"
  },
  {
    "字段": "createTime",
    "类型": "日期",
    "排序": "降序"
  }
]
```

### 四、云函数部署

#### 4.1 安装云函数依赖

在微信开发者工具中，右键点击每个云函数文件夹（`cloudfunctions/login` 等），选择「在终端中打开」，执行：

```bash
npm install
```

#### 4.2 上传并部署云函数

依次右键点击以下云函数文件夹，选择「上传并部署：云端安装依赖」：

1. `cloudfunctions/login`
2. `cloudfunctions/addHomework`
3. `cloudfunctions/completeHomework`
4. `cloudfunctions/exchangeReward`
5. `cloudfunctions/generateRecurringTasks`

等待所有云函数部署完成（约2-3分钟）。

#### 4.3 配置定时触发器（可选）

为周期作业生成功能配置定时触发器：

1. 进入云开发控制台 -> 云函数 -> `generateRecurringTasks`
2. 点击「触发器」->「添加触发器」
3. 配置如下：
   - 触发周期：每天
   - 触发时间：00:00
   - Cron 表达式：`0 0 0 * * * *`

### 五、小程序上传和发布

#### 5.1 修改项目配置

打开 `project.config.json`，修改 `appid` 为你的小程序 AppID：

```json
{
  "appid": "你的小程序AppID"
}
```

#### 5.2 上传代码

1. 在微信开发者工具中，点击右上角「上传」
2. 填写版本号（如：1.0.0）
3. 填写项目备注（如：首次发布）
4. 点击「上传」

#### 5.3 提交审核

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「管理」->「版本管理」
3. 在「开发版本」中找到刚上传的版本
4. 点击「提交审核」
5. 填写审核信息：
   - 功能页面：首页、添加作业、作业打卡、奖励商城
   - 测试账号：填写可用于测试的微信号
6. 提交审核

#### 5.4 发布上线

1. 审核通过后，进入「审核版本」
2. 点击「发布」
3. 确认发布信息
4. 小程序正式上线

### 八、常见问题解决

#### 8.1 云函数调用失败

**问题**：调用云函数时报错 "云函数不存在"

**解决**：
1. 检查云函数是否已部署
2. 检查云函数名称是否正确
3. 查看云函数日志定位问题

#### 8.2 数据库权限错误

**问题**：数据库查询时报权限错误

**解决**：
1. 进入云开发控制台 -> 数据库
2. 点击集合名称 -> 「权限设置」
3. 选择「所有用户可读，仅创建者可写」

#### 8.3 图片上传失败

**问题**：上传图片到云存储失败

**解决**：
1. 检查云存储是否已开启
2. 检查存储空间是否充足
3. 检查文件大小是否超过限制

#### 8.4 Canvas 绘制失败

**问题**：分享海报生成失败

**解决**：
1. 检查小程序基础库版本 >= 2.9.0
2. 检查 Canvas ID 是否正确
3. 查看 Console 错误信息

### 九、监控和维护

#### 9.1 云开发监控

定期查看：
- 数据库使用情况
- 云函数调用次数
- 存储空间使用量
- CDN 流量消耗

#### 9.2 日志查看

进入云开发控制台 -> 云函数 -> 查看日志，及时发现和解决问题。

#### 9.3 数据备份

定期导出数据库数据：
1. 进入云开发控制台 -> 数据库
2. 选择集合 -> 「导入/导出」
3. 导出 JSON 格式数据
