# 从聊天导入作业功能说明

## 功能概述

提供一站式作业导入体验,支持从微信聊天、相册或拍照导入作业内容,并使用AI OCR自动识别图片文字。

## 使用方法

### 方式1: 选择/拍照 (推荐)
1. 点击"选择/拍照"按钮
2. 选择"从聊天选择图片"或"拍照识别作业"
3. 如果选择了图片,系统会询问是否使用AI识别
4. 识别成功后自动填充到作业内容

**使用场景**:
- 保存了作业图片到相册
- 直接拍摄作业图片
- 快速导入和识别

### 方式2: 聊天转发
1. 点击"聊天转发"按钮
2. 在微信聊天中长按需要导入的消息
3. 选择"转发" → 点击本小程序
4. 系统自动识别并填充

**使用场景**:
- 老师在群里发了作业图片
- 聊天中有作业相关的文本消息
- 需要直接从聊天导入

## 功能特性

### 智能识别
- **文本消息**: 直接获取并填充
- **图片消息**: 使用AI OCR识别文字内容
- **多图支持**: 一次最多选择9张图片

### 一站式体验
- 选择图片 → 自动识别 → 填充内容 → 提交作业
- 无需在多个页面间跳转
- 流畅的用户体验

### 视觉反馈
- 清晰的操作指引
- 实时图片预览
- 识别进度提示

## 技术实现

### 前端 (pages/add/add.js)
- 提供两种导入方式
- 处理图片选择和拍照
- 调用OCR云函数识别文字
- 自动填充表单字段

### 云函数 (cloudfunctions/ocrGeneral)
- 使用腾讯云OCR通用印刷体识别服务
- 支持识别印刷体文字
- 返回结构化的识别结果

### UI/UX (pages/add/add.wxml + add.wxss)
- 现代化卡片式布局
- 直观的快捷操作按钮
- 清晰的使用说明

## 注意事项

1. **隐私安全**: 小程序无法主动读取聊天记录,必须通过用户主动操作(转发或选择图片)
2. **OCR识别**: 识别准确率取决于图片清晰度和字体清晰度
3. **网络要求**: OCR识别需要联网才能使用
4. **云函数**: 需要先部署 `ocrGeneral` 云函数才能使用图片识别功能

## 云函数部署

### 方法1: 使用微信开发者工具(推荐)
1. 打开微信开发者工具
2. 在左侧文件树中找到 `cloudfunctions/ocrGeneral` 文件夹
3. 右键点击该文件夹
4. 选择 **"上传并部署:云端安装依赖"**
5. 等待部署完成
6. 在云开发控制台的云函数页面确认部署成功

## 示例场景
